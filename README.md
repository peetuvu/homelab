# Homelab

Personal homelab running on Proxmox VE on an Intel Xeon E3-1230 v2 based server.

## Current deployed services
- Home Assistant (VM 100)

## Planned next services
- Vaultwarden
- Tailscale
- Uptime Kuma
- AdGuard Home

## Hardware
- CPU: Intel Xeon E3-1230 v2
- RAM: 8 GB DDR3 1600
- Storage: 480 GB Kingston SATA SSD
- GPU: GTX 660

## Repository structure
- `docs/` documentation
- `proxmox/` Proxmox node and VM notes
- `homeassistant/` Home Assistant related configs and ideas
- `containers/` service deployment files
- `scripts/` helper scripts and setup tools

## Goals
- Stable home automation base
- Secure remote access
- Lightweight self-hosted core services
- Reproducible infrastructure and documentation
